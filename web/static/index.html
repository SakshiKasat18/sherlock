    📖 Analysis Glossary

    Reference guide to the heuristics and classifications used in this dashboard.

    CIOH — Common Input Ownership

    Multiple inputs in a transaction likely belong to the same wallet, since a single entity must sign all inputs.

    Change Detection

    Identifies outputs likely returning change to the sender using script type matching and round-number analysis.

    CoinJoin Detection

    Equal-value outputs from multiple inputs indicate participants deliberately combining funds for privacy mixing.

    Consolidation

    Many inputs merged into few outputs — standard wallet UTXO cleanup, often done during low-fee periods.

    Address Reuse

    The same scriptPubKey appears in both inputs and outputs, linking UTXO sets and reducing on-chain privacy.

    Round Number Payment

    Output values divisible by round BTC denominations — a signal that a human specified the payment amount.

    Batch Payment

    Few inputs, many outputs — one sender paying multiple recipients simultaneously. Common in exchange withdrawals.

    Classification Categories

    Each transaction is labeled: Simple Payment, Consolidation, CoinJoin, Self Transfer, Batch Payment, or Unknown.
